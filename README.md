# Mini Search Engine

A Python-based GUI search engine that allows you to search YouTube videos and web pages using the YouTube Data API v3 and Google Custom Search API.
This project is built with Tkinter and displays video thumbnails, titles, links, and web search results in a scrollable interface.

## Features

  * Search YouTube videos directly from the app
  * Clickable thumbnails and video links
  * Search web pages using Google Custom Search API
  * Scrollable, user-friendly Tkinter GUI
  * Displays video titles, snippets, and links
